    Sistem Persamaan Linier

    Sistem Persamaan linier merupakan kumpulan dari persamaan-persamaan linier yang memiliki bentuk variabel yang sama. Bentuk umum dari system persamaan linier dengan n peubah dinyatakan dalam bentuk sebagai berikut $$ \begin{aligned}a_{11}x_{1}+a_{12}x_{2}+\ldots +a_{1n}x_{n}=b_{1}\end{aligned} $$

    \[ \begin{aligned}a_{21}x_{1}+a_{22}x_{2}+\ldots +a_{2n}x_{n}=b_{2}\end{aligned} \]
    \[ \begin{aligned}...\end{aligned} \]
    \[ \begin{aligned}A_{m1}x_{1}+a_{m2}x_{2}+\ldots +a_{nn}x_{n}=b_{n}\end{aligned} \]

    Dengan menggunakan perkalian matriks maka kita dapat menulis persamaan diatas sebagai persamaan matriks $$ Ax=b $$

    Dimana

    A = [ai,j] adalah matriks berukuran nxn

    x = [xj] adalah matriks berukuran nx1

    b = [bj] adalah matriks berukuran nx1 (disebut matriks kolom)

    Yaitu

    \[ \begin{bmatrix} a_{11} & a_{12} & a_{13} & \ldots & a_{1n} \\ a_{21} & a_{22} & a_{23} & \ldots & a_{2n} \\ a_{31} & a_{32} & a_{33} & \ldots & a_{3}n \\ a_{n1} & a_{n1} & a_{n1} & \ldots & a_{nn} \end{bmatrix}\begin{bmatrix} x_{1} \\ x_{2} \\ x_{3} \\ \cdot \cdot \\ x_{n} \end{bmatrix}=\begin{bmatrix} b_{1} \\ b_{2} \\ b_{3} \\ \cdot \cdot \\ b_{n} \end{bmatrix} \]

    Tiga kemungkinan solusi SPL

    1. Solusi tunggal = berpotongan pada satu titik
    2. Solusi tak hingga banyaknya = berhimpit
    3. Tidak ada solusi = sejajar (tak berpotongan pada satu titik pun)

    Untuk menemukan solusi penyelesaian system persamaan linier terdapat dua metode yang dapat dilakukan yaitu method langsung yakni mencari system persamaan linier dalam langkah berhingga contohnya method gauss dan gauss Jordan, yang kedua adalah method tak langsung atau methode iterasi seperti Jacobi dan gauss siedel

    Methode Gauss¶

    Teori¶

    Merupakan metode yang dikembangkan dari metode eliminasi yaitu menghilangkan atau mengurangi jumlah variabel tertentu sehingga dapat diperoleh nilai dari suatu variabel bebas.

    Skema dari metode ini yang pertama adalah mengubah persamaan linier kedalam matriks. Kemudian mengubah matriks bagian segitiga bawah kebentuk Eselon Baris melalui forward elimination, setelah itu baru sistem diselesaikan dengan substitusi mundur untuk mendapatkan nilai dari variabel-variabel tersebut

    Eliminasi Maju (Forwad Elimination)

    \[ Eliminasi..Xn_{bi\leftarrow b_{i}-\left( \dfrac{ain}{a_{nn}}\right) b_{2}}^{a_{i1}\leftarrow a_{in},-\left( \dfrac{a_{in}}{a_{nn}}\right) a_{nn}}\left( n\leq j\leq n\right) |n\leq i\leq n \]

    Substitusi Mundur (Back Subtitution)

    \[ \begin{aligned}Xi=\dfrac{b_{i}-\sum ^{n}_{j=i+1}a_{ij},X_{j}}{a_{i,i}}\\ \end{aligned} \]

    Contoh Soal¶

    \[ \begin{aligned}9x_{1}-2x_{2}+x_{3}=11\\ -2x_{1}+4x_{2}-2x_{3}=-16\\ -x_{1}+2x_{2}-4x_{3}=17\end{aligned} \]

    Eliminasi Maju

    Substitusi Mundur

    Algoritma¶

    • Start
    • Deklarasi function gauss dengan paramater matriks a dan b, n = len(b)
    • Perulangan loop dalam loop untuk melakukan forward elimination
    • Jika list a indeks ke i j != 0 maka akan dilakukan operasi menggunakan algoritma forward elimination
    • Perulangan loop for untuk melakukan back subtitution
    • return n
    • stop

    Code¶

    import numpy as np

def gauss(a,b):
  n = len(b)

  #Forward Elimination
  for k in range(0,n-1):
    for i in range(k+1,n):
      if a[i,k] != 0.0:
        lam = a [i,k]/a[k,k]
        a[i,k+1:n] = a[i,k+1:n] - lam*a[k,k+1:n]
        b[i] = b[i] - lam*b[k]

  #Back Substitution
  for k in range(n-1,-1,-1):
    b[k] = (b[k] - np.dot(a[k,k+1:n],b[k+1:n]))/a[k,k]
  return b
a = np.array([[4.0, -2.0, 1.0], [-2.0, 4.0, -2.0], [1.0, -2.0, 4.0]])
b = np.array([11.0, -16.0, 17.0])
print(a)
print(b)
print("============ Hasil dari penyelesaian dengan methode gauss ==========")
print(gauss(a,b))

    Methode Jacobi¶

    Merupakan methode yang termasuk dalam metode penyelesaian secara tak langsung yaitu bermula dari suatu hampiran penyelesaian awal dan kemudian berusaha memperbaiki hampiran dalam tak berhingga namun langkah kovergen. Metode ini cocok untuk menyelesaikan persamaan linier dalam skala besar

    Rumus¶

    \[ \begin{aligned}Fori=1:n\\ x_{1}^{\left( k+1\right) }=\left( b_{i}-\sum ^{i-1}_{j=1}a_{j}x_{0}^{\left( k\right) }-\sum ^{n}_{j=i+1}a_{ij}x_{j}^{k}\right) /a_{ii}\\ end\end{aligned} \]

    Contoh Soal¶

    \[ \begin{aligned}x_{1}-2x_{2}+3x_{3}=10\\ 2x_{1}+3x_{2}+3x_{3}=13\\ x_{1}+x_{2}+x_{3}=5\end{aligned} \rightarrow \begin{aligned}x_{1} = (10+2x_{2}+3x_{3})/1\\x_{2} = (2x_{1}+3x_{3}-13)/3\\x_{3} = (x_{1}+x_{2}-5)/1\end{aligned} \]
    \[ \begin{aligned}iterasi-1\rightarrow x_{1}=0,x_{2}=0,x_{3}=0\\\\ x_{1} = (10+2(0) + 3(0))= 10\\ x_{2} = (2(0)+3(0)-13)/3=4.33\\ x_{3}= (0+0-5)=5 \\\\\\ iterasi-2\rightarrow x_{1}=10,x_{2}=4.33,x_{3}=5\\\\ x_{1} = (10+2(4.33) + 3(5))= -13.66\\ x_{2} = (2(10)+3(5)-13)/3=-7.33\\ x_{3}= (10+4.33-5)=9.33\\\\ dst.. \end{aligned} \]

    Algoritma¶

    • Function jacobi parameter list matriks dan jumlah interasi
    • Buat list blist2 = [] untuk menyimpan hasil dari tiap iterasi
    • Perulangan loop dalam loop untuk melakukan operasi sesuai dengan rumus jacobi
    • perulangan berhenti ketika jumlah iterasi sesuai dengan inputan
    • Tampilkan hasil
    • Stop

    Code¶

    def jacobi(aList, bList, cList, iterasi):
  bList2 = []
  for it in range(iterasi):
    print('Iterasi' , it + 1)
    for i in range(len(aList)):
      hasil = 0
      for j in range(len(aList[0])):
        if i != j :
          if it != 0 :
            bList[j] = bList2[j]
          hasil += aList[i][j] * -1 * bList[j]
      c = (cList[i] + hasil ) / aList[i][i]
      print('X',i+1,c)
      bList2.append(c)
    if it != 0 :
      del bList2[0:i+1]

    Methode Gauss-Seidel¶

    Teori¶

    Metode yang menggunakan proses iterasi hingga diperoleh nilai yang berubah-ubah

    Rumus¶

    \[ \begin{aligned}Fori=1:n\\ x_{1}^{\left( k+1\right) }=\left( b_{i}-\sum ^{i-1}_{j=1}a_{j}x_{0}^{\left( k+1\right) }-\sum ^{n}_{j=i+1}a_{ij}x_{j}^{k}\right) /a_{ii}\\ end\end{aligned} \]

    Contoh Soal¶

    \[ \begin{aligned}2x_{1}+3x_{2}-x_{3}=6\\ x_{1}+2x_{2}-4x_{3}=8\\ x_{1}+x_{2}+4x_{3}=4\end{aligned} \rightarrow \begin{aligned}x_{1} = (6-3x_{2}+x_{3})/2\\x_{2} = (8-x_{1}+4x_{3})/2\\x_{3} = (4-x_{1}-x_{2})/4\end{aligned} \]
    \[ \begin{aligned}iterasi-1\rightarrow x_{1}=0,x_{2}=0,x_{3}=0\\\\ x_{1} = (6-3(0) + (0))/2= 3\\ x_{2} = (8-3-4(0)/2=2.5\\ x_{3}= (4-3-2.5)/4=-0.375 \\\\\\ iterasi-2\rightarrow x_{2}=2.5,x_{3}=0.375\\\\ x_{1} = (6-3(2.5) + 0.375)= -0.9375\\ x_{2} = (8-(-0.9375)+4(0.375))/3=3.718\\ x_{3}= (4-(-0.9375)-3.718)/4=0.304\\\\ dst.. \end{aligned} \]

    Algoritma¶

    • Function gaussSiedel parameter list matriks dan jumlah interasi
    • Perulangan loop dalam loop untuk melakukan operasi sesuai dengan rumus gauss-seidel
    • pemberian nilai bList[i] = c untuk menyesuaikan nilai x1,x2,x3 dalam setiap iterasi
    • perulangan berhenti ketika jumlah iterasi sesuai dengan inputan
    • Tampilkan hasil
    • Stop

    Code¶

    def gaussSiedel(aList, bList, cList, iterasi):
  for it in range(iterasi):
    print('Iterasi' , it + 1)
    for i in range(len(aList)):
      hasil = 0
      for j in range(len(aList[0])):
        if i != j :
          hasil += aList[i][j] * -1 * bList[j]
      c = (cList[i] + hasil ) / aList[i][i]
      bList[i] = c
      print('X',i+1,' : ',c)

    Differensial Biasa

    Persamaan diferensial biasa (Ordinary Differential Equation) merupakan bentuk dari persamaan diferensial yang hanya mempunyai satu variabel bebas. Jika y(x) adalah suatu fungsi satu variabel, maka x dinamakan variabel bebas dan y dinamakan variabel tak bebas

    Contoh persamaan dari diferensial biasa

    \[ \dfrac{dy}{dx}=6x+5\\ \]
    \[ e^{y}\dfrac{d^{2}y}{dx^{2}}+a\left( \dfrac{dy}{dx}\right) ^{2}=1\\ \]
    \[ 2\dfrac{d^{3}y}{dx^{2}}+\sin \times \dfrac{d^{2}y}{dx^{2}}+axy=0\\ \]

    Contoh diatas merupakan contoh dari persamaan diferensial biasa (PDB) dimana fungsi y yang tidak diketahui dan terdiri hanya pada variabel independen x

    Dalam menyelesaikan persoalan PDB terdapat banyak cara yang dapat dilakukan, salah satunya dengan menggunakan methode paling fundamental yaitu Methode Euler

    Methode Euler¶

    Teori¶

    Methode Euler merupakan methode paling sederhana yang diturunakan dari deret Tylor meskipun dalam penyelesaiannya sering menghasilkan error yang cukup besar. Namun methode ini lah yang paling sederhana dalam proses penyelesaiannya

    Dalam methode euler proses penyelesaian dilakukan dengan mencari nilai fungsi y(x) pada titik x tertentu dari persamaan diferensial biasa f(x,y) yang diketahui dengan menggunakan persamaan umum

    Rumus¶

    \[ y_{i+1}=y_{i}+f\left( x_{i},y_{i}\right) h \]

    dimana i merupakan tahapan iterasi

    Contoh Soal¶

    \[ \begin{aligned}Y^{'}=2xy+1\\ Y\left( 0\right) =2,h=0.1\end{aligned} \]

    Jawaban

    \[ \begin{aligned}n =0\rightarrow y_{1}=y_{0}+hf\left( x_{0},y_{0}\right) \\ =2+0,1^{*}\left( 2^{*}0^{*}2+1\right) \\ =2.1\end{aligned} \]
    \[ \begin{aligned}n =1\rightarrow y_{2}=y_{1}+hf\left( x_{1},y_{1}\right) \\ =2+0,1^{*}\left( 2^{*}0.1^{*}2.1+1\right) \\ =2.242\end{aligned} \]

    Algoritma¶

    • Function funcEuler parameter x, y berisi persamamaan yang akan diselesaikan
    • Function euler parameter n sebagai iterasi, h = selang, x = nilai x0, y = nilai y0
    • Membuat listy = [] menyimpan hasil dari tiap iterasi sebelumnya
    • perulangan loop for untuk menghitung operasi sesuai rumus euler
    • tampilkan hasil
    • stop

    Code¶

    def funcEuler(x,y):
  a = 2*x*y + 1 # Y = 2xy + 1
  return a
def euler(n,h,x,y):
  listy = []
  listy.append(y)

  # euler method
  for i in range(n):
    result = listy[i] + (h*funcEuler(x,y))
    listy.append(result)
    x += h
    print("y",i, " : ",result)

x = int(input("Kondisi Awal x0 : "))
y = int(input("Kondisi Awal y0 : "))
h = float(input ("Nilai h : "))
n = int(input("Jumlah Iterasi : "))
euler(n,h,x,y)
